<br/></br>### 课程介绍:
本套课程主要探讨如何读懂 C/C++ 语言程序的反汇编代码。所谓反汇编，简单的讲就是把可执行文件的二进制编码翻译成汇编语言代码，从汇编代码中读懂原高级语言的含义。这是一套基础课程，着重讲解反汇编代码与 C/C++ 语法的对应关系，抓住最核心的知识点。
